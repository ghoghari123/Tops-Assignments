Static vs Absolute Position

WhatsApp Chat Example

A static element stays in the normal page flow, so it takes up space and pushes nearby content. When the notification badge is changed to absolute, it is removed from that normal flow and placed exactly inside its parent container. In apps like WhatsApp, this lets a small unread-count badge sit on the top-right corner of a chat card without pushing the chat name or message preview away.
